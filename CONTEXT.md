# Nurim — Domain Glossary

## Core
사용자 자신을 나타내는 중심 노드. 분자 구조의 가장 큰 원.
각 사용자는 하나의 Core를 가진다.

## Orbit
Core 주변에 붙는 목표 노드. 운동, 공부, 연애 등 자기개발 카테고리.
사용자가 자유롭게 추가/삭제할 수 있다.
Orbit은 두 가지 유형을 가진다:
- **Accumulation Orbit**: 기간 내 누적 목표 (예: 이번 달 100km)
- **Repetition Orbit**: 매일 반복 목표 (예: 매일 5km)

## Pulse
하루에 한 번 Orbit에 찍는 기록. 숫자 단위로 입력하며 소급 입력이 허용된다.
사용자 양심에 기반한다 — 시스템은 강제하지 않는다.

## Proof
사용자가 자신의 분자 그래프(Core + Orbits)를 커뮤니티 피드에 공유하는 게시물.
기본은 비공개이며, 사용자가 명시적으로 공유할 때만 피드에 노출된다.
